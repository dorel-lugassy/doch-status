# 📊 מערכת דוחות אקסל

> אפליקציית Streamlit לעיבוד וסיווג הזמנות אינטרנט מורכב להרצה

---

## מבנה הפרויקט

```
דוח סטטוס חדש/
├── main.py                   ← ממשק המשתמש (Streamlit)
├── processors/
│   └── internet_morchav.py   ← לוגיקת עיבוד הנתונים
└── utils/
    └── excel_utils.py        ← עזרים לטעינה ושמירת Excel
```

---

## זרימת עבודה

```
קובץ Excel (3 גיליונות)
        │
        ▼
   load_sheets()
   ┌────────────────────────────┐
   │  סיבים │ נחושת │ כל השאר  │
   └────────────────────────────┘
        │
        ▼
  internet_morchav.run()
        │
        ├──▶ טבלת סטטוס אינטרנט   → קובץ Excel להורדה
        ├──▶ טבלת חריגים (אם יש)  → גיליון נוסף באותו קובץ
        └──▶ טבלת קו טלפון        → קובץ Excel נפרד
```

## לוגיקת עיבוד

### 1 · שורה אחרונה לכל הזמנה

גיליונות **סיבים** ו-**נחושת** מאוחדים. לכל `מספר הזמנה` — נשמרת **השורה האחרונה בלבד**.  
התאריכים נלקחים מאותה שורה.

---

### 2 · סיווג סטטוס ההזמנה

הסטטוס נקבע **אך ורק** לפי עמודת **סטטוס שירות**:

| סטטוס שירות | סטטוס בפלט | תאריך שנלקח |
|:-----------:|:----------:|:-----------:|
| `סגור` | **סגור** | תאריך סגירת הזמנה |
| `פתוח` | **פתוח** | תאום לקוח התחלה |
| `מבוטל` | **מבוטל** | — |
| ערך אחר | **חריג** | — |

> שורות עם ערך לא מוכר ב"סטטוס שירות" עוברות לגיליון **חריגים** בקובץ הפלט.

---

### 3 · שורות מיוחדות מגיליון "כל השאר"

#### BIZNET
שורות שבהן `שירות = BIZNET` — מתווספות לטבלת האינטרנט הראשית.  
התאריך נלקח מ: `פרטי תאום לקוח במשימה אחרונה בהוראת עבודה`.

#### PHONE
שורות שבהן `שירות = PHONE` **ויש ערך ב-`חבילת דקות`** — יוצרות דוח קו טלפון נפרד.

## עמודות קלט נדרשות

### גיליונות סיבים / נחושת

| עמודה | תפקיד |
|-------|--------|
| `מספר הזמנה` | מזהה ייחודי לקיבוץ |
| `מספר הכר` | מזהה כרטיס |
| `סטטוס שירות` | קובע את סטטוס ההזמנה |
| `תאום לקוח התחלה` | תאריך להזמנות פתוחות |
| `תאריך סגירת הזמנה` | תאריך להזמנות סגורות |

### גיליון כל השאר

| עמודה | תפקיד |
|-------|--------|
| `מספר הזמנה` | שיוך לכרטיס |
| `מספר הכר` | מזהה כרטיס |
| `שירות` | סינון BIZNET / PHONE |
| `שירות מלא` | זיהוי נקודות WIFI לביטולי Mesh |
| `סטטוס שירות` | סטטוס לשורות BIZNET / PHONE |
| `פרטי תאום לקוח במשימה אחרונה בהוראת עבודה` | תאריך מתואם |
| `חבילת דקות` | סינון שורות PHONE |

> **שים לב:** שורת הכותרות בקובץ נמצאת ב**שורה 2** (שורה 1 היא כותרת ראשית).
